Install Parcel :

# `npm install --save-dev parcel`

---

Install Parcel-bundler:

# `npm install parcel-bundler --save-dev`

Install dependencies:

# `npm install`

---

To start a project :

## `npm run dev`

---

To watch live server:

# `http://localhost:1234`
